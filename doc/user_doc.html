1. What is EMPub?

Empub is an extension of the standard mailer. Using this application, a mail user can easily share a message from his inbox, providing access to it to multiple friends in just one click.

2. Getting started

2.1 Publish your first email!

  • Redirect the email you want to publish from your email software to our address :
    empub.send@gmail.com
  • Publish parameters can be set through a tag system included in the body of the email, for example: [Password = mysecretpassword]
    [Duration = 1y2mon4h5m7s]
  • The default publishing parameters are:
    [public]
    [Duration = 1mon]

2.2 Login

If you forwarded at least one email to our address, the account associated with your email address has been created.
Otherwise, if you just want to see other publications, you can log in using a valid email address.

In either case, a temporary password will be sent to the email address you entered.

EMPub application

3.1 Using the homepage

On the homepage you can see all your published emails, the date of their publication / expiration, whether public or private, or the number of views.

  • You can filter them with permission (public / private), you can sort them by publication date, expiration date, or number of views.
  • You can search for emails using the search bar (keywords will be searched in the sender's name and subject).
  • You can view the content of the emails
  • You can delete emails

3.2 Using email settings page

On this page you can see the publishing parameters and update them. You can also delete the email.

3.3 Using the email statistics page

On this page you can view statistics related to a publication that belongs to you (total views, number of views grouped by country of origin of visitors) during the selected period (last 24 hours, last week, last month).
Also, these statistics can be downloaded in xml or pdf formats.

3.4 Using the account settings page

On this page you can view the email address associated with your account, you can view and update the first and last name associated with your profile.

3.4 Using the publishers page

On this page you can view and search for other publishers. You can also see the total number of their publications.

3.4 Using the publisher page

On this page you can see all the emails published by another publisher, the date of their publication / expiration. You can search, filter and sort these emails.
In order to read private publication, you must enter the password set by the publisher.
